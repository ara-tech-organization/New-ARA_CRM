# Implementation Plan: Google Ads Analytics + Billing CRM Extension

## Overview
Extend the existing Node.js/Express/MongoDB CRM system with Google Ads analytics and billing functionality. The current codebase has a solid foundation with client management, leads, and basic reporting - we'll build upon this to add multi-client Google Ads integration.

This guide provides detailed step-by-step instructions for beginners. We'll explain concepts as we go and include code examples.

## Prerequisites & Concepts

### Required Credentials
- **Google Ads API access**: Developer token, OAuth2 client ID/secret, refresh token
- **Google Ads Manager Account**: With access to client accounts
- **Environment variables**: For secure credential storage

### Key Concepts
- **Google Ads API**: Programmatic interface to fetch campaign data, metrics, etc.
- **MongoDB Collections**: Like tables in a database for storing data
- **Mongoose Models**: JavaScript classes that represent MongoDB collections
- **Express Routes**: API endpoints for frontend to fetch data
- **Cron Job**: Scheduled task that runs automatically (e.g., every 6 hours)
- **Micros**: Google Ads uses micros (millionths) for currency - need to convert to rupees/paise

### Tools Needed
- Node.js installed
- npm (comes with Node.js)
- Google Cloud Console access
- Google Ads account with API access

### India-Specific Considerations
- Default currency set to INR (Indian Rupees)
- Default time zone set to Asia/Kolkata (IST)
- Ensure your Google Ads Manager Account has access to Indian client accounts
- Google Ads API policies may vary for Indian advertisers - review compliance requirements

## Detailed Setup Guide

### Phase 0: Google Ads API Setup (Most Important Step)

Since you mentioned not having enough idea, let's start with the API setup in detail.

#### Step 1: Get Google Ads Developer Token
1. Go to [Google Ads API Center](https://ads-developers.googleblog.com/)
2. Sign in with your Google Ads account
3. Apply for API access (may take 24-48 hours)
4. Once approved, get your developer token from the API Center

#### Step 2: Create Google Cloud Project
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create new project or select existing
3. Enable Google Ads API:
   - Go to "APIs & Services" > "Library"
   - Search for "Google Ads API"
   - Click "Enable"

#### Step 3: Create OAuth2 Credentials
1. In Google Cloud Console: "APIs & Services" > "Credentials"
2. Click "Create Credentials" > "OAuth 2.0 Client IDs"
3. Choose "Web application"
4. Set authorized redirect URIs to: `http://localhost` (for local testing)
5. Download the JSON file - this contains client_id and client_secret

#### Step 4: Install Google Ads Library
```bash
npm install google-ads-api
```

#### Step 5: Get Refresh Token
Create a script to get your refresh token:

```javascript
// get-refresh-token.js
import { GoogleAdsApi } from 'google-ads-api';
import readline from 'readline';

const client = new GoogleAdsApi({
  client_id: 'YOUR_CLIENT_ID',
  client_secret: 'YOUR_CLIENT_SECRET',
  developer_token: 'YOUR_DEVELOPER_TOKEN'
});

const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

const authUrl = client.getOAuthUrl();
console.log('Visit this URL to authorize:', authUrl);

rl.question('Enter the authorization code: ', (code) => {
  client.getRefreshToken(code).then((refreshToken) => {
    console.log('Refresh token:', refreshToken);
  }).catch((error) => {
    console.error('Error:', error.message);
  });
  rl.close();
});
```

Run it and follow the prompts to get your refresh token.

#### Step 6: Environment Variables
Add to your `.env` file:
```
GOOGLE_ADS_CLIENT_ID=your_client_id
GOOGLE_ADS_CLIENT_SECRET=your_client_secret
GOOGLE_ADS_DEVELOPER_TOKEN=your_developer_token
GOOGLE_ADS_REFRESH_TOKEN=your_refresh_token
```

### Core Integration Module
Create `backend/services/googleAdsService.js`:

```javascript
const { GoogleAdsApi } = require('google-ads-api');

class GoogleAdsService {
  constructor() {
    this.client = new GoogleAdsApi({
      client_id: process.env.GOOGLE_ADS_CLIENT_ID,
      client_secret: process.env.GOOGLE_ADS_CLIENT_SECRET,
      developer_token: process.env.GOOGLE_ADS_DEVELOPER_TOKEN
    });
  }

  // Create customer instance for a specific client
  getCustomer(customerId) {
    return this.client.Customer({
      customer_id: customerId,
      refresh_token: process.env.GOOGLE_ADS_REFRESH_TOKEN
    });
  }

  // Convert micros to currency (Google Ads uses micros for precision)
  convertMicrosToCurrency(micros) {
    return micros / 1000000; // 1,000,000 micros = 1 currency unit
  }

  // Aggregate click types
  aggregateClickBreakdown(clickTypeData) {
    const breakdown = { website_clicks: 0, call_clicks: 0, other_clicks: 0 };

    clickTypeData.forEach(item => {
      const clicks = item.metrics.clicks || 0;
      switch(item.segments.click_type) {
        case 'WEBSITE':
          breakdown.website_clicks += clicks;
          break;
        case 'CALLS':
          breakdown.call_clicks += clicks;
          break;
        default:
          breakdown.other_clicks += clicks;
      }
    });

    return breakdown;
  }

  async fetchCampaigns(customerId) {
    const customer = this.getCustomer(customerId);
    const campaigns = await customer.query(`
      SELECT
        campaign.id,
        campaign.name,
        campaign.status,
        campaign_budget.amount_micros
      FROM campaign
      WHERE campaign.status != 'REMOVED'
    `);

    return campaigns.map(c => ({
      campaign_id: c.campaign.id,
      name: c.campaign.name,
      status: c.campaign.status,
      budget: this.convertMicrosToCurrency(c.campaign_budget.amount_micros)
    }));
  }

  async fetchMetricsWithClicks(customerId, dateRange = 'LAST_30_DAYS') {
    const customer = this.getCustomer(customerId);

    // Fetch metrics with click types
    const data = await customer.query(`
      SELECT
        campaign.id,
        campaign.name,
        metrics.impressions,
        metrics.clicks,
        metrics.cost_micros,
        metrics.conversions,
        segments.click_type,
        segments.date
      FROM campaign
      WHERE segments.date DURING ${dateRange}
      AND campaign.status != 'REMOVED'
    `);

    // Group by campaign and date, then aggregate click types
    const grouped = {};
    data.forEach(row => {
      const key = `${row.campaign.id}_${row.segments.date}`;
      if (!grouped[key]) {
        grouped[key] = {
          campaign_id: row.campaign.id,
          campaign_name: row.campaign.name,
          date: row.segments.date,
          impressions: 0,
          clicks: 0,
          cost: 0,
          conversions: 0,
          click_types: []
        };
      }

      grouped[key].impressions += row.metrics.impressions || 0;
      grouped[key].clicks += row.metrics.clicks || 0;
      grouped[key].cost += this.convertMicrosToCurrency(row.metrics.cost_micros || 0);
      grouped[key].conversions += row.metrics.conversions || 0;
      grouped[key].click_types.push({
        click_type: row.segments.click_type,
        metrics: row.metrics
      });
    });

    // Convert to array and add click breakdown
    return Object.values(grouped).map(item => ({
      ...item,
      click_breakdown: this.aggregateClickBreakdown(item.click_types)
    }));
  }
}

module.exports = new GoogleAdsService();
```

### Data Fetchers
- Methods in the service class above handle fetching
- Use GAQL (Google Ads Query Language) for queries
- Handle pagination automatically (library does this)

### Data Transformers
- Built into the service class
- Convert micros: divide by 1,000,000
- Aggregate clicks by type

## Phase 3: Sync Service Implementation

### Cron Job Setup
Install node-cron:
```bash
npm install node-cron
```

Create `backend/sync/syncService.js`:

```javascript
const cron = require('node-cron');
const Client = require('../models/Client');
const Campaign = require('../models/Campaign');
const Metric = require('../models/Metric');
const googleAdsService = require('../services/googleAdsService');

class SyncService {
  constructor() {
    // Schedule to run every 6 hours (adjust as needed)
    cron.schedule('0 */6 * * *', () => {
      console.log('Starting Google Ads sync...');
      this.syncAllClients();
    });
  }

  async syncAllClients() {
    try {
      const clients = await Client.find({ /* add filter for clients with Google Ads */ });

      for (const client of clients) {
        await this.syncClient(client);
      }

      console.log('Sync completed for all clients');
    } catch (error) {
      console.error('Sync failed:', error);
    }
  }

  async syncClient(client) {
    try {
      console.log(`Syncing client: ${client.name} (${client.customer_id})`);

      // 1. Fetch and store campaigns
      const campaigns = await googleAdsService.fetchCampaigns(client.customer_id);
      for (const campaignData of campaigns) {
        await Campaign.findOneAndUpdate(
          { campaign_id: campaignData.campaign_id },
          {
            client_id: client._id,
            ...campaignData
          },
          { upsert: true, new: true }
        );
      }

      // 2. Fetch and store metrics (last 30 days)
      const metrics = await googleAdsService.fetchMetricsWithClicks(client.customer_id);
      for (const metricData of metrics) {
        await Metric.findOneAndUpdate(
          {
            client_id: client._id,
            campaign_id: metricData.campaign_id,
            date: new Date(metricData.date)
          },
          metricData,
          { upsert: true, new: true }
        );
      }

      // 3. Update client billing info
      await this.updateClientBilling(client._id);

      console.log(`Sync completed for client: ${client.name}`);
    } catch (error) {
      console.error(`Sync failed for client ${client.name}:`, error);
      // Could implement retry logic here
    }
  }

  async updateClientBilling(clientId) {
    // Calculate total spend from metrics
    const totalSpendResult = await Metric.aggregate([
      { $match: { client_id: clientId } },
      { $group: { _id: null, total: { $sum: '$cost' } } }
    ]);

    const totalSpend = totalSpendResult[0]?.total || 0;

    // Update client
    await Client.findByIdAndUpdate(clientId, {
      'billing.total_spend': totalSpend,
      'billing.available_balance': /* total_added_funds - total_spend */
    });
  }

  // Manual trigger method for testing
  async manualSync(clientId = null) {
    if (clientId) {
      const client = await Client.findById(clientId);
      if (client) await this.syncClient(client);
    } else {
      await this.syncAllClients();
    }
  }
}

module.exports = new SyncService();
```

### How to Run the Sync
In your main server file (`backend/server.js`):
```javascript
const syncService = require('./sync/syncService');
// This will start the cron job automatically
```

For testing, you can call:
```javascript
// In a route or manually
syncService.manualSync(); // sync all
syncService.manualSync(clientId); // sync specific client
```

### Sync Process Flow
1. **Load Clients**: Query DB for clients with Google Ads enabled
2. **For Each Client**:
   - Fetch campaigns from Google Ads API
   - Store/update in campaigns collection (upsert)
   - Fetch metrics with click breakdown
   - Store/update in metrics collection (upsert)
   - Recalculate total spend and balance
3. **Error Handling**: Log errors, could add retry logic
4. **Rate Limiting**: Google Ads API has limits - the library handles this

## Phase 4: API Extensions

### Example Route Creation
Create `backend/routes/metrics.js`:
```javascript
const express = require('express');
const router = express.Router();
const Metric = require('../models/Metric');

// Get aggregated metrics for dashboard
router.get('/dashboard/:clientId', async (req, res) => {
  try {
    const { clientId } = req.params;
    const { startDate, endDate } = req.query;

    const metrics = await Metric.aggregate([
      {
        $match: {
          client_id: clientId,
          date: {
            $gte: new Date(startDate),
            $lte: new Date(endDate)
          }
        }
      },
      {
        $group: {
          _id: null,
          totalClicks: { $sum: '$clicks' },
          totalCost: { $sum: '$cost' },
          totalConversions: { $sum: '$conversions' },
          websiteClicks: { $sum: '$click_breakdown.website_clicks' },
          callClicks: { $sum: '$click_breakdown.call_clicks' }
        }
      }
    ]);

    res.json(metrics[0] || {});
  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});

module.exports = router;
```
