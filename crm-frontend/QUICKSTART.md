# Quick Start Guide

Get your CRM application running in 3 simple steps!

## Step 1: Install Dependencies

```bash
npm install
```

## Step 2: Start the Application

```bash
npm start
```

The app will open automatically at `http://localhost:3000`

## Step 3: Login

Use these credentials:

- **Email**: `admin@crm.com`
- **Password**: `admin123`

---

## What's Included?

### Pages
- **Dashboard** - Overview with stats, charts, and today's leads
- **Clients** - Manage your client database
- **Leads** - Track and manage sales leads

### Features
- Modern, responsive UI with Material-UI
- State management with Redux
- Form validation
- Data visualization with charts
- Secure authentication
- Protected routes
